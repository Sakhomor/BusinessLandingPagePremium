# BusinessLandingPagePremium — SNE Portfolio

## Description
Ce projet est une **landing page premium** pour présenter les réalisations de **SNElegance**.  
Le site est entièrement responsive et utilise HTML, CSS et JavaScript.

Il met en avant :
- **SN Élégance** — site e-commerce pour la vente de tissus Getzner.
- **Portfolio professionnel** — pour présenter les projets web et designs.
- **Atelier Carrosserie** — site vitrine pour services et galerie.

Toutes les images sont **locales** dans le dossier `images/` pour assurer la compatibilité et la vitesse de chargement sur GitHub Pages.

## Technologies utilisées
- HTML5
- CSS3
- JavaScript
